# Chapter4

## 名詞介紹
- Protocol 通訊協定
- Router 路由器
- Gateway 閘通道
- IEEE 電機電子工程師學會
- TCP/IP Transmission Control Protocol / Internet Protocol
- OSI -> Open System Interconnection

## OSI 分層
1. 應用層    Application Layer
2. 呈現層    Presentation Layer
3. 會議層    Session Layer
4. 傳輸層    Transport Layer 
5. 網路層    Network Layer
6. 資料連結層 Data Link Layer
7. 實體層    Physical Layer

- 軟體：一到六
- 韌體：三到六
- 硬體：五到七
